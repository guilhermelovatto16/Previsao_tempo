# Previsão do Tempo

App simples que mostra o clima de qualquer cidade.  
Feito com HTML, CSS e JavaScript usando a API do OpenWeather.

## Como usar

1. Abra o `index.html`.  
2. Digite o nome da cidade.  
3. Clique em buscar.
